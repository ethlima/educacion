# Convertirte en contributor

## Requisitos
- Haber hecho al menos 1 PR aceptado
- Seguir el estilo de redacción
- Respetar el código de conducta

## Proceso
1. Abre un issue presentándote
2. Indica qué sección mejorarás
3. Entrega PR
4. Se te agrega a Contributors
