# Learning Paths — Ethereum Lima Academy

Rutas de aprendizaje con temas especificos por nivel. Para charlas mensuales de builders, cursos en la plataforma, y medir interes de la comunidad.

## Tabla general de niveles

| Nivel | Experiencia | Que logras | Tiempo estimado |
|---|---|---|---|
| **Basico** | 0-3 meses | Entender fundamentos, primer deploy en testnet | 3-4 semanas |
| **Intermedio** | 3-9 meses | Construir dApps completas, deploy en Arbitrum | 4-6 semanas |
| **Avanzado** | 9+ meses | Protocolos, auditorias, arquitecturas complejas | 6-8 semanas |

## Conexion con el programa "Sembrando Juntos"

```
Nivel Basico          Nivel Intermedio         Nivel Avanzado
(pre-bootcamp)   →    (bootcamp sem 1-3)  →    (bootcamp sem 4-6 + hackathon)

Fundamentos            Construccion             Especializacion
Blockchain, Solidity   dApps, Frontend, L2      DeFi, Stylus, Seguridad
```

Los niveles justifican las **actividades previas al bootcamp**: dado que Arbitrum es complejo y no muchos lo conocen, los participantes necesitan completar al menos el nivel Basico antes de que arranque "La Siembra".
